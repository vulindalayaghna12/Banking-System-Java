# Banking System (Core Java)

## Description
This project is a console-based banking system developed using Core Java. It allows users to perform basic banking operations.

## Features
- User Registration
- Login Authentication
- Deposit and Withdrawal
- Fund Transfer
- Transaction History

## Technologies Used
- Java
- OOP Concepts
- ArrayList

## How to Run
1. Compile Main.java
2. Run the program
3. Follow menu options
